# Focus Compass – Vezetői Stratégiai Fókusz Tervező

Interaktív stratégiai fókusz-tervező eszköz vezetőknek, a Diamond Coaching gondozásában.

## Tartalom

- `index.html` — a teljes alkalmazás egyetlen önálló HTML fájlban (Tailwind CDN, jsPDF, html2canvas)

## Telepítés

Nincs telepítési lépés. A fájl statikus, bármelyik böngészőben működik, és a Vercel automatikusan kiszolgálja.

## Használat

Nyisd meg az `index.html` fájlt, vagy látogasd meg az élő verziót:
**https://diamondcoaching-focus.vercel.app**

## Frissítés

A repo `index.html` fájlját módosítva a Vercel automatikusan újradeployol.
